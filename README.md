# Lane

一個多功能Bot，還是繁體中文!，並且還有`dashboard`! <br>

# 安裝 [ Windows ]

```
git clone https://github.com/Tira-tw/Lane
```
```
cd Lane
```
```
npm install
```

# 安裝 [ hosting ] 

[下載Lane](https://github.com/Tira-tw/Lane/archive/refs/heads/main.zip)


Startup > ADDITIONAL NODE PACKAGES

```
npm install
```
